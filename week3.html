Working With Divs

A div stands for division. The div tag defines a section in an html document. You can use divs as well to section off your HTML and organize them. You can also give divs names for better organization and structure. Here's an example.




Basically what's going on in that DIV is that whatever is inside, is changing the text color to red. You can use the properties I spoke of before to edit this.


Divs are usually also used to make boxes, add padding and margins and things like that. But there are a few specifics that we must know before we go on to this.

There are two types of Div's. Div ID's and Div Classes. Div ID's are set divs that can only be used ONCE. These are specified in CSS by using a '#'. For example, you can use this for a navigation, or a title of the webpage that will only be used once! Div Classes can be used multiple times, and are specified in CSS using a '.' YOU CANNOT START AN ID OR A CLASS WITH A NUMBER!
